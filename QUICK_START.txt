📌 ขั้นตอนที่ 1: ติดตั้ง Ubuntu และ Docker
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. คลิกขวาที่ไฟล์: setup_wsl_docker.bat
2. เลือก "Run as administrator"
3. รอให้ Ubuntu ติดตั้งเสร็จ (5-10 นาที)
4. Ubuntu terminal จะเปิดขึ้นมา
5. ตั้งชื่อผู้ใช้และรหัสผ่าน


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📌 ขั้นตอนที่ 2: ติดตั้ง Docker ใน Ubuntu
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

ใน Ubuntu terminal ที่เปิดอยู่ ให้รันคำสั่งนี้:

    cd /mnt/g/เทสทวิต
    chmod +x install_docker_in_ubuntu.sh
    ./install_docker_in_ubuntu.sh

รอให้ติดตั้งเสร็จ (10-15 นาที)


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📌 ขั้นตอนที่ 3: เริ่มระบบ
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

หลังติดตั้ง Docker เสร็จ:

1. ปิด Ubuntu terminal แล้วเปิดใหม่
2. รันคำสั่ง:

    cd /mnt/g/เทสทวิต
    chmod +x start_news_bot.sh
    ./start_news_bot.sh

3. เปิดเบราว์เซอร์:

    http://localhost:5678

4. Login:

    Username: admin
    Password: admin123


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✅ เสร็จแล้ว!
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

ตอนนี้คุณสามารถ:

1. Import workflow:
   - ใน n8n คลิก "Import from File"
   - เลือก: Flow 2 โพสต์ข่าว.json

2. ตั้งค่า Credentials:
   - Google Sheets OAuth2
   - OpenRouter API
   - Facebook Graph API

3. Activate workflow:
   - คลิกสวิตช์ "Active"

4. ทดสอบ:
   - เพิ่มงาน "pending" ใน Google Sheets
   - รอ 2 นาที
   - เช็คว่าโพสต์ไปยัง Facebook แล้ว


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📚 คำสั่งที่ใช้บ่อย
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

เปิด Ubuntu terminal แล้วรัน:

# เริ่มระบบ
cd /mnt/g/เทสทวิต
docker-compose up -d

# ดูสถานะ
docker-compose ps

# ดู logs
docker-compose logs -f
docker-compose logs -f n8n
docker-compose logs -f youtube-downloader-api

# หยุดระบบ
docker-compose down

# Restart
docker-compose restart

🆘 ถ้ามีปัญหา
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

   → ลองติดตั้งจาก Microsoft Store
   → ค้นหา "Ubuntu 22.04"

2. Docker ติดตั้งไม่ได้:
   → ตรวจสอบ internet connection
   → ลองรันสคริปต์อีกครั้ง

3. Containers ไม่เริ่ม:
   → docker-compose down
   → docker-compose up -d
   → docker-compose logs -f

⚠️  สถานะปัจจุบัน: Docker ยังไม่ได้ติดตั้ง
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━


📌 ขั้นตอนที่ 1: ติดตั้ง Docker Desktop (จำเป็น!)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. เปิดลิงก์นี้ในเว็บเบราว์เซอร์:
   
   👉 https://www.docker.com/products/docker-desktop/
   
2. คลิก "Download for Windows"
   
3. เมื่อดาวน์โหลดเสร็จ ให้ติดตั้ง Docker Desktop:
   - Double-click ไฟล์ที่ดาวน์โหลด
   - เลือก "Use WSL 2 instead of Hyper-V" (ถ้าถาม)
   - คลิก "Ok" และรอให้ติดตั้งเสร็จ
   
4. ⚠️  RESTART WINDOWS (สำคัญมาก!)
   
5. หลัง restart เสร็จ:
   - เปิดโปรแกรม "Docker Desktop"
   - รอจน Docker Engine พร้อม (ไฟเป็นสีเขียว)
   - กด "Accept" ใน License Agreement


📌 ขั้นตอนที่ 2: ตรวจสอบว่า Docker พร้อมใช้งาน
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

หลังติดตั้ง Docker เรียบร้อยแล้ว:

วิธีที่ 1 (ง่าย):
   📂 Double-click ไฟล์: check_system.bat
   
วิธีที่ 2 (PowerShell):
   1. เปิด PowerShell
   2. พิมพ์: docker --version
   3. ถ้าเห็นเลขเวอร์ชัน แสดงว่าพร้อมแล้ว!


📌 ขั้นตอนที่ 3: เริ่มระบบ
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

เมื่อ Docker พร้อมแล้ว:

วิธีที่ 1 (แนะนำมากที่สุด!):
   📂 Double-click: menu.bat
   
วิธีที่ 2:
   📂 Double-click: start.bat
   
วิธีที่ 3 (PowerShell):
   cd "g:\เทสทวิต"
   docker-compose up -d

1. เปิดเว็บเบราว์เซอร์ไปที่:
   🌐 http://localhost:5678
   
2. Login:
   👤 Username: admin

   - เลือกไฟล์: Flow 2 โพสต์ข่าว.json

📌 คำสั่งที่ใช้บ่อย
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

ดู logs:         docker-compose logs -f
Restart:         docker-compose restart
หยุดระบบ:        docker-compose down


📌 ปัญหาที่พบบ่อย

❌ วิดีโอดาวน์โหลดไม่ได้
   → docker-compose logs -f youtube-downloader-api

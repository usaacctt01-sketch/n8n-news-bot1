# ⚡ Quick Start Guide - เริ่มใช้งานด่วน

## 📋 เช็คลิสต์ก่อนเริ่ม

- [ ] ติดตั้ง Python แล้ว (3.8+)
- [ ] ล็อกอิน Twitter ใน Chrome แล้ว
- [ ] รู้ Chrome Profile Path
- [ ] เตรียมรูปภาพ/วิดีโอไว้แล้ว
- [ ] ปิด Chrome ทั้งหมดแล้ว

---

## 🚀 เริ่มใช้งาน 3 ขั้นตอน

### 1️⃣ ติดตั้ง
ดับเบิลคลิก `run_bot.bat` หรือรันใน Command Prompt:
```bash
pip install -r requirements.txt
```

### 2️⃣ รันโปรแกรม
ดับเบิลคลิก `run_bot.bat` หรือ:
```bash
python twitter_bot.py
```

### 3️⃣ ตั้งค่า
ใส่ข้อมูลในโปรแกรม:
- Chrome Profile Path
- เพิ่มไฟล์รูป/วิดีโอ
- ใส่แฮชแท็ก
- กำหนดจำนวนและเวลา
- กด **Start Posting**

---

## 🎯 Tips สำหรับผู้เริ่มต้น

### ✅ ควรทำ
- ทดสอบด้วยจำนวนน้อยๆ ก่อน (1-2 โพส)
- ตั้งเวลาอย่างน้อย 10 นาที
- ใช้รูปภาพหลากหลาย
- เช็คว่าล็อกอิน Twitter แล้ว

### ❌ ไม่ควรทำ
- โพสเร็วเกินไป (< 5 นาที)
- ใช้ไฟล์ขนาดใหญ่เกินไป
- ลืมปิด Chrome ก่อนรัน
- ใส่ Path ผิด

---

## 🔍 วิธีหา Chrome Profile Path แบบเร็ว

1. เปิด Chrome
2. พิมพ์: `chrome://version/`
3. คัดลอก "Profile Path"
4. ตัด `\Default` ออกจากท้าย

**ดูรายละเอียดใน:** `CHROME_PROFILE_GUIDE.md`

## 🆘 แก้ปัญหาด่วน

| ปัญหา | วิธีแก้ |
|-------|---------|
| Chrome ไม่เปิด | ปิด Chrome ให้หมด |
| Path ไม่ถูกต้อง | ใช้ `chrome://version/` |
| อัพโหลดไม่ได้ | ตรวจสอบขนาดไฟล์ |
| โพสไม่สำเร็จ | ตรวจสอบว่าล็อกอิน Twitter |
| Error ตอนรัน | ติดตั้ง selenium ใหม่ |

## 📝 Checklist ก่อนกด Start

1. ✓ ปิด Chrome ทั้งหมด
2. ✓ ใส่ Chrome Profile Path
3. ✓ เพิ่มไฟล์อย่างน้อย 1 ไฟล์
4. ✓ ใส่แฮชแท็ก (ถ้าต้องการ)
5. ✓ ตั้งจำนวนโพส
6. ✓ ตั้งช่วงเวลา (≥ 5 นาที)
7. ✓ กด Start Posting!
